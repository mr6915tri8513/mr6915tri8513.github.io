這是一個校內競賽的專題。概念和名稱是由 sstteeeenn05 想的，而我負責 APP 製作。

## 學習成果
### Arduino
* ESP32 的經典藍芽
* DrawerLayout

### Android
* 經典藍芽

### 其他
* Google 圖表 (折線圖)
* Base64

## 簡介
這個項目可以分成兩個部分：手機上的 **APP** 和 **電競檢測儀** (用盒子裝起來的那個)

將檢測儀裝到該裝的地方之後，在手機上以藍芽設定完就可以使用了，簡單方便。
